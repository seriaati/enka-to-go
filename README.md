# enka-to-go

[>> Website <<](https://enka-to-go.seria.moe)

Export your character showcase from [Enka Network](https://enka.network) to JSON formats compatible with [Genshin Optimizer](https://frzyc.github.io/genshin-optimizer/) (GOOD) or [Zenless Optimizer](https://frzyc.github.io/zenless-optimizer/) (ZOD).
